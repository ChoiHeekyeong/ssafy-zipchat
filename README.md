# ssafy-zipchat
### SSAFY 8기 1학기 최종 관통프로젝트

<br>

## **개요**
- 국토 교통부 아파트 매매 실거래 자료를 통해 쉽게 시세를 알 수 있게 해주고 주변 상권 및 시설 위치를 함께 제공, 관심 지역을 설정하여 관심 지역의 정보를 추가 제공

<br>

## 프로젝트 기간
- 2021년 11월 16일 ~ 2021년 11월 24일
